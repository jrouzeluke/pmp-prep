# BATCH 1: Lead a Team - Complete Content Package

## Overview

This package contains all content for BATCH 1 of the project management training program. The module focuses on team leadership, covering Tuckman's stages of team development, leadership styles, and practical application through engaging activities.

---

## Package Contents

### Learn Hub Materials (Theory & Context)

1. **1_overview.md** - Module Introduction
   - Learning objectives
   - Why this matters
   - Success statistics
   - Module structure

2. **2_pmp_connection.md** - PMP Application Connection
   - People domain alignment (42% of exam)
   - Specific task mappings
   - Application tips
   - How to document experience

3. **3_tuckmans_model.md** - Deep Dive: Tuckman's Stages
   - Forming, Storming, Norming, Performing, Adjourning
   - Characteristics of each stage
   - Team member behaviors
   - PM leadership actions
   - Common pitfalls
   - Duration and triggers
   - Key insights and practical application
   - ~8,000 words, comprehensive reference

4. **4_leadership_styles.md** - Deep Dive: Leadership Styles
   - Six leadership styles (Goleman model)
   - When to use each style
   - Strengths and weaknesses
   - Situational leadership framework
   - Style combinations
   - Common mistakes
   - Self-assessment tools
   - ~7,500 words, comprehensive reference

---

### Activities (Hands-On Application)

5. **5_activity_pm_simulator.md** - PM Simulator Lightning Round
   - Duration: 20-30 minutes
   - Format: Individual → Small Group
   - 10 realistic project scenarios
   - Practice rapid diagnosis and decision-making
   - Answer key with rationale
   - Discussion questions

6. **6_activity_document_detective.md** - Document Detective
   - Duration: 30-40 minutes
   - Format: Individual Analysis → Group Discussion
   - 6 project artifacts to analyze
   - Identify team stages from written clues
   - Compare effective vs ineffective leadership
   - Pattern recognition practice

7. **7_activity_timeline_reconstructor.md** - Timeline Reconstructor
   - Duration: 30-40 minutes
   - Format: Small Group Collaborative
   - 12 event cards to arrange chronologically
   - Map team development over 9-month project
   - Identify leadership interventions and impacts
   - Turning point analysis

8. **8_activity_relationship_matcher.md** - Relationship Matcher (Leadership Styles)
   - Duration: 20-30 minutes
   - Format: Individual → Pairs → Group
   - 18 situations to match with leadership styles
   - Practice situational judgment
   - Decision tree framework
   - Style selection practice

9. **9_activity_empathy_exercise.md** - Empathy Exercise
   - Duration: 30-40 minutes
   - Format: Individual Reflection → Pair → Group
   - 5 team member perspectives
   - Deep perspective-taking
   - Emotionally engaging
   - Leadership action planning

## Content Structure

### Learn Hub Components

The Learn Hub should present content in this order:

1. **Overview** (from 1_overview.md)
   - Hook learners with compelling introduction
   - Set clear expectations
   - Establish relevance

2. **PMP Connection** (from 2_pmp_connection.md)
   - For PMP candidates: Show exam relevance
   - For others: Skip or present as "Professional Certification Connection"
   - Make it easy to find but not mandatory

3. **Deep Dive: Tuckman's Model** (from 3_tuckmans_model.md)
   - Core conceptual framework
   - Should be primary reference material
   - Support with visuals/diagrams if possible

4. **Deep Dive: Leadership Styles** (from 4_leadership_styles.md)
   - Second core conceptual framework
   - Complements Tuckman's model
   - Include self-assessment tools

### Activity Sequence (Recommended Order)

**Suggested Flow:**

1. **PM Simulator Lightning Round** (5_activity_pm_simulator.md)
   - START HERE - Quick wins, build confidence
   - Activates prior knowledge
   - Low stakes, high engagement

2. **Relationship Matcher** (8_activity_relationship_matcher.md)
   - Reinforces leadership styles
   - Pattern recognition
   - Builds on Simulator momentum

3. **Document Detective** (6_activity_document_detective.md)
   - Deeper analysis required
   - Real-world application
   - Pattern recognition in artifacts

4. **Timeline Reconstructor** (7_activity_timeline_reconstructor.md)
   - Synthesizes Tuckman's model
   - Team collaboration required
   - Complex but rewarding

5. **Empathy Exercise** (9_activity_empathy_exercise.md)
   - END HERE - Most emotionally engaging
   - Deep reflection
   - Personal action planning
   - Leaves lasting impact

**Alternative: Choose Your Own Adventure**
- Allow learners to select activities based on interest
- Provide 2-3 "must do" + 2-3 "choose one"
- Accommodate different learning styles

   - Read thoroughly
