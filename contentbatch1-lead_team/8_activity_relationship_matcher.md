# Activity: Relationship Matcher (Leadership Styles)

## Activity Overview

**Duration:** 20-30 minutes  
**Format:** Individual → Pairs → Group Discussion  
**Difficulty:** Easy-Medium  
**Learning Objectives:**
- Connect leadership styles to appropriate situations
- Understand when each style is most/least effective
- Practice situational judgment
- Build leadership style flexibility

---

## Instructions

You'll receive two sets of cards:

**Set A: Leadership Style Cards** (6 cards)
- Commanding
- Authoritative
- Affiliative
- Democratic
- Pacesetting
- Coaching

**Set B: Situation Cards** (18 cards)

**Your Tasks:**

1. **Individual (5 min):** Match each situation to the BEST leadership style
2. **Pairs (10 min):** Compare answers with a partner, discuss differences
3. **Group (10 min):** Discuss challenging matches and reasoning
4. **Debrief (5 min):** Review answers and key principles

---

## Discussion Questions

### Pattern Recognition

1. **Which situations had the clearest answers?** Why?
   - Crises typically need Commanding
   - Development needs typically need Coaching
   - Complex decisions typically need Democratic

2. **Which situations were most debatable?** What made them ambiguous?
   - Multiple styles could work
   - Context matters (team maturity, urgency, etc.)
   - Style combinations often most effective

3. **Did you notice any situations where multiple styles could work?**
   - Examples and how to choose

### Application Thinking

4. **Which situations are most common in your projects?**
   - How do you currently handle them?
   - What might you do differently?

5. **Which styles do you use most naturally?**
   - Which do you avoid?
   - Why?

6. **What prevents you from using certain styles?**
   - Comfort level?
   - Skills?
   - Organizational culture?
   - Time?

### Deeper Analysis

7. **Why is Affiliative style appropriate for both exhausted team (Situation 3) and conflict (Situation 9)?**
   - Both need relationship focus
   - Different applications of same style

8. **Why use Commanding for crisis (Situation 7) but Democratic for complex decision (Situation 4)?**
   - Time pressure vs. time available
   - Expertise distribution
   - Risk factors

9. **When would Pacesetting be inappropriate despite time pressure?**
   - Team burnout
   - Skill gaps
   - Low trust
   - Sustained vs. short-term

### Personal Development

10. **Which style do you need to develop?**
    - What's stopping you?
    - How could you practice?

11. **Think of a situation you handled recently. What style did you use?**
    - Was it effective?
    - What would you do differently?

12. **How can you build flexibility across styles?**
    - Practice opportunities
    - Feedback sources
    - Development resources

## Style Selection Decision Tree

Use this framework when facing real situations:

```
START: What's the situation?

├─ EMERGENCY/CRISIS?
│  └─ Yes → **COMMANDING** (temporarily)
│  └─ No → Continue...
│
├─ TEAM EXHAUSTED/STRAINED?
│  └─ Yes → **AFFILIATIVE** (prioritize recovery)
│  └─ No → Continue...
│
├─ NEED CLEAR DIRECTION/VISION?
│  └─ Yes → **AUTHORITATIVE** (inspire and guide)
│  └─ No → Continue...
│
├─ COMPLEX DECISION + TIME + EXPERTISE IN TEAM?
│  └─ Yes → **DEMOCRATIC** (collaborative decision)
│  └─ No → Continue...
│
├─ SHORT-TERM PUSH + CAPABLE TEAM?
│  └─ Yes → **PACESETTING** (temporary excellence drive)
│  └─ No → Continue...
│
├─ DEVELOPMENT OPPORTUNITY?
│  └─ Yes → **COACHING** (build capabilities)
│  └─ No → Continue...
│
└─ HIGH-PERFORMING TEAM + ROUTINE WORK?
   └─ **DELEGATING** (empower and trust)
```

## Facilitator Notes

**Setup:**
- Print situation cards (one set per participant or pair)
- Create style reference sheet
- Prepare answer key
- Plan for disagreements (they're valuable!)

**Time Management:**
- Individual matching: 5-7 minutes
- Pair discussion: 10 minutes
- Group discussion: 10-15 minutes
- Debrief and key principles: 5 minutes

**Facilitation Tips:**
- Multiple valid answers often exist
- Focus on reasoning, not memorization
- Encourage debate and discussion
- Connect to participants' experiences
- Emphasize style flexibility

**Common Patterns to Highlight:**
- **Crisis → Commanding:** Always appropriate temporarily
- **Development → Coaching:** Long-term capability building
- **Complex + Time → Democratic:** Leverage expertise
- **Inspiration needed → Authoritative:** Provide vision
- **Recovery needed → Affiliative:** Prioritize relationships
- **Short push → Pacesetting:** Temporary only
- **High performance → Delegating:** Trust and empower

**Adaptations:**
- **Beginner:** Provide style definitions with cards
- **Advanced:** Remove answer key, have groups justify choices
- **Virtual:** Use polling or breakout rooms
- **Large group:** Work in teams of 3-4
