# CURSOR MASTER INSTRUCTIONS: PMP Learn Hub - All 35 Tasks (7-5-7 Format)

## Overview

This document is the **master reference** for building the PMP Learn Hub. It defines:
- All 35 ECO tasks organized by domain
- **7-5-7 structure** for Learn Hub tabs (Overview: 7, PMP Application: 5, Deep Dive: 7)
- Activity structure for each task (standard vs. specialized)
- File naming conventions
- Content templates for all content types
- Implementation guidance

**Content will be added task by task.** This document provides the structure and templates.

---

# PART 1: THE 7-5-7 STRUCTURE

## Learn Hub Tabs: Standard Section Counts

| Tab | Sections | Purpose | User Time |
|-----|----------|---------|-----------|
| **Overview** | 7 | Quick reference card | 5 min |
| **PMP Application** | 5 | Exam patterns + practice | 10-15 min |
| **Deep Dive** | 7 | Comprehensive study | 30-45 min |

---

## Overview: 7 Sections

| # | Section Name | Content |
|---|--------------|---------|
| 1 | **What This Task Is** | Definition + Core Principle (combined) |
| 2 | **ECO Enablers** | Official PMI enablers (bulleted list) |
| 3 | **Key Frameworks** | Primary models/tools for this task (tables, visuals) |
| 4 | **How To Apply It** | Practical application examples, best practices |
| 5 | **Exam Triggers** | Phrases/situations that signal this task |
| 6 | **Quick Scenarios** | 4-5 Right/Wrong with 1-line explanations |
| 7 | **Common Mistakes** | Wrong answer patterns to avoid on exam |

---

## PMP Application: 5 Sections

| # | Section Name | Content |
|---|--------------|---------|
| 1 | **How This Task Is Tested** | Intro paragraph + question patterns table |

| 3 | **Sample Question 2** | Full scenario + 4 options + analysis |
| 4 | **Sample Question 3** | Full scenario + 4 options + analysis |

---

## Deep Dive: 7 Sections

| # | Section Name | Content |
|---|--------------|---------|
| 1 | **Introduction** | Why this matters, what you'll learn |
| 2 | **[Primary Framework]** | Deep explanation of main framework |
| 3 | **[Secondary Framework/Topic]** | Second major concept |
| 4 | **Practical Application** | How to use in real projects |
