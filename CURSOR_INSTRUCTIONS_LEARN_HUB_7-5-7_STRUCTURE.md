# PMP LEARN HUB - CURSOR INSTRUCTIONS

## PROJECT OVERVIEW
Building a PMP certification training platform with 35 task modules. Each task follows the **7-5-7 structure** for Learn Hub content plus 6 interactive activities.

## THE 7-5-7 STRUCTURE

### Overview Tab: 7 Sections
1. **What This Task Is** - Definition + Core Principle
2. **ECO Enablers** - Official PMI enablers
3. **Key Frameworks** - Primary models/tools
4. **How To Apply It** - Practical examples
5. **Exam Triggers** - Phrases that signal this task
6. **Quick Scenarios** - 4-5 Right/Wrong with 1-line why
7. **Common Mistakes** - Wrong answer patterns

### PMP Application Tab: 5 Sections
1. **How This Task Is Tested** - Intro + patterns table
2. **Sample Question 1** - Full scenario + analysis
3. **Sample Question 2** - Full scenario + analysis
4. **Sample Question 3** - Full scenario + analysis
5. **Exam Strategy** - Do/Don't + key phrases

### Deep Dive Tab: 7 Sections
1. **Introduction** - Why this matters
2. **[Primary Framework]** - Deep explanation
3. **[Secondary Framework]** - Second concept
4. **Practical Application** - Real-world usage
5. **Common Challenges** - What goes wrong
6. **Connections to Other Tasks** - Related tasks
7. **Key Takeaways** - 8-10 principles
